# **PostV4Ordermodify**
> OrderModifyResponse PostV4Ordermodify (OrderModifyRequest? orderModifyRequest = null)

Modify an Existing Order

The order modify transaction allows for changes to be made after the order creation process but before the order is released to Ingram Microâ€™s warehouse system.  Order modify transaction submitted after the order is released will be rejected and will not be applied.  Types of modifications allowable: Order release, add comment, and carrier change.
